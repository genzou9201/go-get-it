# go-get-it

A simple Go application that encourages you to "go get it"!

## Installation

```bash
go mod download
```

## Usage

Run the application:

```bash
go run main.go
```

Or with a custom message:

```bash
go run main.go "your goals"
```

## Building

```bash
go build -o go-get-it
./go-get-it
```
